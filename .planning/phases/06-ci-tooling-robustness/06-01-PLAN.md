# Phase 6 Plan 01 — CI / Tooling Robustness (CICD-07, CICD-08)

<objective>
Make the SPDX/REUSE gate and the gsd-sdk phase-plan-index parser both run clean. Pin the
reuse encoding dependency (CICD-07) and bring the repo back to REUSE compliance by licensing
GSD planning artifacts via REUSE.toml instead of inline headers (CICD-08), so PLAN.md
frontmatter can stay on line 1 for the parser. Document the convention.
</objective>

<threat_model>
Trust boundary: CI workflow + repo governance metadata only; no runtime/app surface, no
network input, no secrets. LAN-only no-auth v1 unchanged.
- (Tampering) Over-broad REUSE.toml glob silently licensing a real source file that lost its
  inline header → CICD-06 regression. Mitigation: scope the new glob to `.planning/**` planning
  artifacts only; never `api/**`/`ui/src/**`. Verified by reuse staying able to flag a
  header-less real source.
- (Integrity) reuse encoding crash masking real compliance failures on CI. Mitigation: pin
  `--with charset-normalizer` so the gate actually executes and reports.
No HIGH threats (config/docs only).
</threat_model>

<tasks>

<task id="1" name="Pin reuse encoding dep in CI (CICD-07)">
<read_first>
- .github/workflows/ci.yml (the `spdx · reuse lint` step, ~line 100-101)
</read_first>
<action>
In `.github/workflows/ci.yml`, change the `spdx · reuse lint` step's `run:` from
`uvx reuse lint` to `uvx --with charset-normalizer reuse lint`. Add a brief comment that the
encoding dep is required (Windows + slim runners lack libmagic; bare reuse raises
NoEncodingModuleError).
</action>
<acceptance_criteria>
- `.github/workflows/ci.yml` contains `uvx --with charset-normalizer reuse lint`
- No remaining bare `uvx reuse lint` invocation in the workflow
</acceptance_criteria>
</task>

<task id="2" name="License planning artifacts via REUSE.toml + gitignore uat-shots (CICD-08)">
<read_first>
- REUSE.toml (existing annotation structure + the CICD-06 philosophy comment)
- .gitignore
</read_first>
<action>
In `REUSE.toml`, add a new `[[annotations]]` block (precedence `aggregate`,
`SPDX-FileCopyrightText = "2026 Brave Bear Studios"`, `SPDX-License-Identifier =
"AGPL-3.0-or-later"`) covering `.planning/**/*.md`, `.planning/**/*.json`, and
`.planning/**/.gitignore`, with a comment documenting the convention: planning artifacts are
licensed here (NOT inline) so `*-PLAN.md` keeps YAML frontmatter on line 1 for the gsd-sdk
phase-plan-index parser; real source files keep their inline two-line header (CICD-06).
In `.gitignore`, add `ui/uat-shots/` (ephemeral UAT screenshots).
</action>
<acceptance_criteria>
- REUSE.toml has an annotation covering `.planning/**/*.md` and `.planning/**/*.json`
- REUSE.toml does NOT add any `api/**` or `ui/src/**` source glob
- `.gitignore` contains `ui/uat-shots/`
- The convention is stated in a REUSE.toml comment
</acceptance_criteria>
</task>
