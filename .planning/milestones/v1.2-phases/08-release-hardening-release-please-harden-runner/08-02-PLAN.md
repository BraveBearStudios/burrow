|----------|-------------|
| upstream GitHub Action → ci.yml/release.yml runners | A repointed mutable action tag injects attacker code into the build/publish jobs |

| PR-title gate action (fork PRs) → CI | The gate runs on pull_request including from forks; a moving-tag pin widens the supply-chain surface |

Every other `uses:` across ci.yml/release.yml is already a 40-hex pin and stays so; the cross-file regex gate asserts no `@vN` survives. |
